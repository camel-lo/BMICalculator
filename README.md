視窗程式設計(II) 上課範例

1.可以透過輸入身高(cm)、體重(kg)，按下計算按鈕後，會顯示BMI值和BMI分類。  
2.在輸入格裡有用灰色的提示文字可以提示使用者如何使用以及避免錯誤。  
3.按下Tab可以切換要輸入的格子(cm、kg、計算)，且按下Enter可以直接計算，減少滑鼠使用。  
4.當Tab到或滑鼠點到該輸入格後會直接把整串文字全選，避免需要一直刪除再輸入。  
5.網頁右邊有圖片顯示你的BMI範圍介於哪個區間，讓你更好去判斷分析。
